# ADO.NET CRUD Management System

A desktop application developed using **C# and ADO.NET** with **SQL Server** for database management.  
The project demonstrates how to build a complete **CRUD system (Create, Read, Update, Delete)** using direct database connectivity through ADO.NET.

## Features
• Connect to SQL Server database using ADO.NET  
• Perform full CRUD operations  
• Simple and user-friendly interface  
• Data validation and error handling  
• Efficient database queries using SQL commands  

## Technologies Used
• C# (.NET)  
• ADO.NET  
• SQL Server  
• Windows Forms  

## Purpose of the Project
This project was developed as a practical implementation of database connectivity in **.NET applications**, focusing on how ADO.NET interacts with SQL Server to manage and manipulate data efficiently.

## What I Learned

• Executing SQL queries using ADO.NET  
• Implementing CRUD operations  
• Managing data between application and database
